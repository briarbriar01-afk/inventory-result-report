- ✅ ناونیشانی نوێ: **ڕاپۆرتی جەردی کەل و پەل**

- 📤 **ناردن**: فۆرمەکەت بۆ سوپەر ئاداین دەنێردیت

- داتای تێکۆشان بە پێی کاتدا
- ئاماری جێبەجێکردن

## 🔄 ڕۆژی کاری کارمەندان


### 2. **فۆرمی جەردی درست کردن**
1. **➕ جەردی نوێ درست بکە** - دەسکا دا بزە
2. زانیاری پڕ بکە:
   - ناوی ئورگان
   - ناوی ژمێریار
   - ڕێکەوت
   - کاتە کان
   - بەرپرسیاران

- **💾 پاشکۆکی**: فۆرمەکەت سێیاتێت (بدەیتە هۆما)

### 4. **چاپ یان PDF**
- **🖨️ چاپ**: بە A4 چاپ دەکەت
- **پس/PDF**: پاشکۆتێپشک بۆ Google Chrome یا Firefox

## 📁 فایلەکانی سیستەم

```
inventory-app/
├── data/
│   ├── users.json          # بەکارهێنەران
│   ├── categories.json     # بەشەکان
│   ├── items.json         # کەلوپەلی
│   └── reports.json       # دراپۆڵی جەردیە ✨ نوێ
├── templates/
│   ├── new_report_form.html       # فۆرمی جەردی نوێ (نوێ کرا)
│   └── superadmin_dashboard.html  # داشبۆردی سوپەر ئاداین ✨ نوێ
└── app.py                  # سِرڤەر (نوێ کراو)
```

---

## 🎯 API Endpoints (بۆ پەرەپێداچونی پێشکەوتی)

### بۆ کارمەندان:
- `POST /api/reports/save` - پاشکۆ لە سەرڤەر
- `POST /api/reports/submit` - ناردن بۆ سوپەر ئاداین

### بۆ سوپەر ئاداین:
- `GET /superadmin-dashboard` - داشبۆرد
- `GET /api/reports/data` - داتای تێکۆشان (ڕابردوو)
