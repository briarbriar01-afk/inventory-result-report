# ✅ سیستەمی جەردی کەل و پەل - چەندەسەی نوێکاری

## 🎯 کردار ئەنجام دراوەکان

### 1. **داشبۆردی سوپەر ئاداین سادەکرا** ✨
- **پێشتر:** خشتە زۆرە ئامار و ڕێوت کور
- **ئێستا:** خشتە سادەو ڕوون:
  - لیستی جزابی هەموو دراپۆڵە
  - دوگمە **🔍 نیشاندان** بۆ هەریەک
  - لە کۆتایدا کۆی جیاوازی نیشات دەدات

### 2. **دوگمەکان کارا کرا** ✅
- **💾 پاشکۆکی:** فۆرمت پاشکۆ دەکات سەرڤەردا
- **🖨️ چاپ:** A4 فرمتی پرینت دەکات
- **📤 ناردن:** بۆ سوپەر ئاداین دەنێردیت
- **📊 نیشاندانی چارت:** چارت نیشان دەدات

### 3. **ناوتێگران نوێ کرا** 
تێپێربس:
```
پێشتر              → ئێستا
├ بەرپرسیارانی جەرد    → بەرپرسیارانی جەردی ئۆرگان
├ بەرپرسی یەکەمی      → بەرپرسی یەکەمی ئۆرگان
├ بەرپرسی دارایی      → بەرپرسی دارایی ئۆرگان
└ بەرپرسی موڵك و ماڵ  → بەرپرسی موڵك و ماڵی ئۆرگان
```

### 4. **فۆرمی جەردی سادەتر و بهتەر کرا**
- جیاوازی **خۆکار** درێژ دەکرێت (ئاد ناچینە!)
- فیڵدەکان **تێپێوی** شەمسن
- پڕینت **مڵیے** لە A4

## 📊 بەرۆڤەی ڕاستی

### **کاریگەر (Employee) ڕۆژی کاری**

1. **پێچنین**
   - بەکارهێنەر: `employee`
   - وشە: `employee123`

2. **فۆرم درست بکە**
   - **➕ جەردی نوێ درست بکە** بزە
   - زانیاری و ژمارە نیسیت

3. **سەرسپاری کردار**
   - **💾 پاشکۆکی** ← سیڤ بکە (بۆ دۆتەر)
   - **🖨️ چاپ** ← PDF یا پڕینت بکە
   - **📤 ناردن** ← بۆ سوپەر ئاداین بنێرە

### **سوپەر ئاداین (Superadmin) ڕۆژی کاری**

1. **پێچنین**
   - بەکارهێنەر: `superadmin`
   - وشە: `superadmin123`

2. **داشبۆرد ببینە**
   - **📊 داشبۆردی سوپەر ئاداین** بزە

3. **دراپۆڵ نیشاندان**
   - لۆ لە خشتەدا **🔍 نیشاندان** بزە
   - ئەنجام نیشان دەبێت (جزئیات)
   - **🖨️ چاپ** دەتوانیت چاپ یا سیڤ بکە

## 🔧 ڕێزدانەکانی دێڕی پۆل (Technical Notes)

### داتای پاشکۆ
- **فایل:** `data/reports.json`
- **فرمت:** JSON (ڕێژیم)
- **دێڕی:** تێکۆشان، ڕێکەوت، ئەنجام

### API Endpoints
```
POST   /api/reports/save      - پاشکۆ بکە (draft)
POST   /api/reports/submit    - بنێرە (submitted)
GET    /superadmin-dashboard  - داشبۆرد
```

### فایلەکانی نوێ
- ✅ `templates/superadmin_dashboard.html` - داشبۆردی نوێ (سادە تر)
- ✅ `templates/new_report_form.html` - فۆرمی ڕووژکار (بړگەکان)
- ✅ `data/reports.json` - دراپۆڵە
